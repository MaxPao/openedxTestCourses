USING Non-Supported EXCEL 2013 (NOT PROFESSIONAL PLUS OR STANDALONE EDITION)

This is the worst possible version to follow the course with. You can’t create any measure and you are pretty much out of luck. The best option in this case is to use a machine on which you can install a trial of Office 2016 preferably the pro-plus version. You can still build simple data models like in the second module and use Excel tables/regions or import data from databases directly into the data model using Excel classic data import capabilities

  • If you use Excel 2013 other than Professional Plus or Standalone edition, you cannot use the Power Pivot add-in. 
  • You can download Power Query and use it with some reduced functionality. Using Power Query, you can add new tables to the data model or edit existing queries.
  • You should be able to open a solution that contains a data model and interact with the pivot tables, pivot chart, slicers etc.
