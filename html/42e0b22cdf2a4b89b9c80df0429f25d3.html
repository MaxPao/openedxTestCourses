Using Non-Supported Excel 2013

Credit to LeonidasT, a fellow learner from the previous run of the course.

"I just found a way to go around the problem and do the exercises of Module 2 just fine. I have an Excel 2013 and no PowerPivot. If you are interested try the following. It's a little cumbersome but you'll enjoy the result!"

  1. Open the file normally. As expected a blank sheet appears with the Data Model hidden.
  2. Go to: Insert > Pivot Table > Use an external data source > Choose connection and open the only connection available in the workbook (the Data Model)
    Now you have a blank Pivot Table connected to the Data Model.
  3. Left-click outside the Pivot Table in order to take away the Pivot Table fields on the right-hand side.
  4. Go to: Data > Existing Connections and like in step 2, open the only connection available. You are asked how to import the data. Select Table > OK and all the tables from the Data Models are imported in Excel, each table in a separate Sheet.
  5. The new sheets with the DM tables are named: Sheet2, Sheet3 etc. Go back to Sheet1, left-click on the empty PivotTable and copy the table names to the corresponding newly created sheets. One good thing is that the new sheets are created with the same order you see the tables in the Pivot Table Fields on the right hand.
  6. Now you have all the necessary data, but the tables are not linked (relationships). Let's fix that by opening: Data > Relationships. In the window you see all the relationships between the tables in the Data Model. What you want is to create relationships between the NEW tables in Excel. So click New and create the new relationships needed, just by copying the ones from the Data Model.
    Check out that the new tables have a "Table_" prefix before the name. Don't get mixed up the the Data Model tables. From now on you forget about them and you work with your new model in Excel, not Power Pivot.
  7. You're done. Now you can do as instructed in the course and the exercise and complete the lab.
