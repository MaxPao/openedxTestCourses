To make your report look like the lab format you simply need to change the report layout from Show in Outline Form to Show in Tabular Form. While the pivot table is active, select the Design tab (I'm using Excel 2013 so your's might be slightly different), from there you will see a Report Layout icon. Click on that icon and you will see a drop down list - select Show in Tabular Form and your report will look just like the screen capture posted in the lab.

You will notice that this changes both the number of columns (one each now for Category and Manufacturer) and changes the location of the subtotals from the top of the group to the bottom of the group. You can also only change the location of the subtotals - from the same Design tab, select the Subtotals Icon and you will see the options available there.
