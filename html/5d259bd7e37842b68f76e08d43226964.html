Step by Step (Excel 2013)

Credit to WLitzenberg.

  1. From a blank workbook, go to the Power Query tab and select the From Database option and then the From SQL Server Database option.
  2. After selecting this option, you will see a SQL Server Database dialog box. Enter the required Server and Database information in the appropriate boxes, then click on the OK button.
    • Server: msedxeus.database.windows.net
    • Database: DAT206x01
  3. Next you will come to a dialog box asking for your Windows credentials to log onto the database - DO NOT use this option - click on Database along the left hand side of the dialog box.

  4. Now enter the provided username and password into the appropriate boxes, then click on the OK button.
    • Username: ExcelLogin
    • Password: P@ssw0rd
  5. This should bring up the Navigator dialog box - click on the Select Multiple Items check box
  6. Then select the four tables to be imported by clicking on the check box next to each: Locations, Manufacturer, Products, and Sales. Click on the Edit button in the bottom right of the dialog box.
  7. You should now be in the query editor and can continue with the lab instructions.
  8. A few more pointers - when you are ready to load the data, choose the Close & Load To... option

  9. Then select the Only Create Connection radio button along with the Add this data to the Data Model check box and then click on the Load button

  10. The answers to the number of rows questions can be found in the resulting Power Query Workbook Queries section on the right hand side of the screen - see below - the correct answers are redacted...
  11. Finally, to review the relationships, click on the Power Pivot tab (I'm using Excel 2013) and then the Manage Data Model icon
  12. From the Power Pivot tab, click on the Diagram View to see the table relationships.
