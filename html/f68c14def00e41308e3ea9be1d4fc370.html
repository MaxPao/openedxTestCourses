Using Non-Supported Excel 2016 (Not Professional Plus or Standalone edition)

If you use Excel 2016 other than Professional Plus or Standalone edition, you don’t have the Power Pivot add-in.

On the other hand, the functionality of Power Query is available in the get-and-transform chunk in the data ribbon. Some advanced Power Query functionality are only available from the pro versions. None of these features is required to the labs in this course.

The features you miss by not having the Power Pivot add-in are:

  • Calculated columns   
  • Automatic calendar creating
  • Hierarchies
  • Hiding tables and columns       
  • Sorting one column by the value in another column
  • Diagram view (but you can manage relationships from a dialog)
  • Categorization of columns
